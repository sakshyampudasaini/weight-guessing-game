# Weight Converter

A simple Python script to convert weight between pounds (lbs) and kilograms (kgs).

## Description

This script prompts the user to enter a weight and specify the unit (pounds or kilograms). It then converts the weight to the other unit and displays the result.

## Features

- Convert pounds to kilograms.
- Convert kilograms to pounds.
- Accepts uppercase or lowercase input for units.

## How to Use

1. Clone or download this repository.
2. Make sure Python is installed on your machine.
3. Run the script using the command:

   ```bash
   python weight_converter.py  ```
